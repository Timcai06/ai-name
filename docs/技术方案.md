## 1. 技术选型

| 层 | 选型 | 版本 | 理由 |
|-----|------|------|------|
| 前端框架 | Vue 3 + Composition API | 3.5 | 开发者熟悉，TypeScript 支持 |

| LLM | DeepSeek API | deepseek-chat | 中文好，便宜 |

| 认证 | JWT (python-jose) | 3.5 | 无状态认证 |
| 密码 | bcrypt (passlib) | 1.7 | 单向哈希 |

## 3. API 接口

### 3.1 认证相关

| 方法 | 路径 | 鉴权 | 说明 |
|------|------|------|------|
| POST | /api/auth/register | 无 | 注册，返回 JWT |
| POST | /api/auth/login | 无 | 登录，返回 JWT |
| POST | /api/auth/logout | JWT | 登出，记录日志 |
| GET | /api/auth/me | JWT | 当前用户信息（含 role） |
| GET | /api/auth/logs | Admin | 认证日志列表 |

### 3.2 取名相关

| 方法 | 路径 | 鉴权 | 说明 |
|------|------|------|------|
| POST | /api/naming/generate | JWT | 生成名字，自动保存历史 |
| POST | /api/naming/refine | JWT | 微调名字，自动保存历史 |
| GET | /api/naming/history | JWT | 个人取名历史（支持分页） |
| GET | /api/naming/history/{id} | JWT | 单条历史详情 |

### 3.3 管理后台

| 方法 | 路径 | 鉴权 | 说明 |
|------|------|------|------|
| GET | /api/admin/namings | Admin | 全部用户的取名记录 |

### 3.4 通用

| 方法 | 路径 | 鉴权 | 说明 |
|------|------|------|------|
| GET | /api/health | 无 | 健康检查 |

## 4. 数据库设计

### 4.1 users

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT PK | 用户 ID |
| username | VARCHAR(50) UNIQUE | 用户名 |
| password_hash | VARCHAR(255) | bcrypt 哈希 |
| role | VARCHAR(20) | user / admin |
| created_at | DATETIME | 注册时间 |

### 4.2 naming_history

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT PK | 记录 ID |
| user_id | INT FK → users.id | 归属用户 |
| surname | VARCHAR(5) | 姓氏 |
| gender | VARCHAR(10) | 性别 |
| birthday | VARCHAR(20) | 生日 |
| birth_time | VARCHAR(10) | 时辰 |
| style | VARCHAR(50) | 风格 |
| expectations | VARCHAR(200) | 期望 |
| result_json | TEXT | 名字结果 JSON |
| feedback | TEXT | 微调对话历史 JSON |
| created_at | DATETIME | 生成时间 |

### 4.3 auth_logs

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT PK | 日志 ID |
| username | VARCHAR(50) | 用户名 |
| action | VARCHAR(20) | register/login/logout |
| ip_address | VARCHAR(45) | 客户端 IP |
| created_at | DATETIME | 操作时间 |

## 5. 权限模型

| 接口 | 未登录 | 普通用户 | 管理员 |
|------|--------|----------|--------|
| register/login | ✅ | ✅ | ✅ |
| generate/refine | 401 | ✅ | ✅ |
| history（自己） | 401 | ✅ | ✅ |
| history（全部） | 401 | 403 | ✅ |
| auth logs | 401 | 403 | ✅ |

- 401：未认证（跳转登录页）
- 403：无权限（跳转主页）

## 6. Prompt 工程

- System Prompt：取名大师角色 + JSON 格式约束 + 禁止值内出现英文双引号
- JSON 解析：三级降级（直接解析 → 提取 code block → 正则匹配 { }）
- 容错处理：自动替换中文弯引号 "" → 「」

## 7. 部署

### 7.1 依赖安装

```bash
cd backend && pip install -r requirements.txt
cd frontend && npm install
```

### 7.2 配置 .env

```ini
DEEPSEEK_API_KEY=sk-xxx
ADMIN_USERNAME=admin
ADMIN_PASSWORD=your_password
```

### 7.3 启动

```bash
# 终端 1
cd backend && uvicorn app.main:app --reload --host 127.0.0.1 --port 8000

# 终端 2
cd frontend && npm run dev
```

### 7.4 切换 MySQL

改 `.env`：`DATABASE_URL=mysql+pymysql://root:password@localhost:3306/naming_db`

SQLAlchemy 代码零改动。

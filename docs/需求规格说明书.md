# 需求规格说明书 — AI 智能取名系统

> 版本：2.0 | 日期：2026-07-03 | 作者：陈奕恺

---

## 1. 引言

### 1.1 项目背景

传统取名依赖长辈经验或翻查字典，效率低且受限于个人知识面。本项目利用大语言模型的语义理解与文化知识储备，为用户提供智能化的取名建议，帮助用户在起名过程中获得更丰富、更专业的选择。

### 1.2 项目范围

本项目为软件外包创新实践基地 2025 级新人实训项目，已完成**难度 1 + 难度 2**。

### 1.3 定义与缩写

| 术语 | 说明 |
|------|------|
| LLM | 大语言模型（Large Language Model） |
| DeepSeek | 本项目使用的大语言模型服务 |
| Prompt | 发送给 LLM 的指令文本 |
| JWT | JSON Web Token，无状态用户认证 |

## 2. 总体描述

### 2.1 用户角色

| 角色 | 说明 | 权限 |
|------|------|------|
| 匿名用户 | 未登录 | 仅登录/注册页面 |
| 普通用户 | 已注册登录 | 取名、微调、查看自己的历史 |
| 管理员 | 系统预设 | 普通用户权限 + 查看认证日志 + 查看全部取名记录 |

### 2.2 核心功能

| 编号 | 功能 | 难度 | 描述 |
|------|------|------|------|
| F-01 | 用户注册 | 2 | 用户名+密码注册，密码 bcrypt 加密存储 |
| F-02 | 用户登录 | 2 | JWT 认证，24小时有效期 |
| F-03 | 取名表单 | 1 | 输入姓氏、性别、生日、时辰、风格、期望 |
| F-04 | AI 名字生成 | 1 | 调 DeepSeek API，返回 5 个名字建议 |
| F-05 | 结果展示 | 1 | 卡片式展示名字、寓意、五行、出处 |
| F-06 | 对话微调 | 1 | 输入反馈意见，系统重新生成 |
| F-07 | 取名历史 | 2 | 查看自己所有历史取名记录 |
| F-08 | 管理后台 | 2 | 管理员查看认证日志+全部取名记录 |
| F-09 | 错误处理 | 1 | AI 超时、网络异常、空结果等状态的友好提示 |

### 2.3 不包含的功能

- 付费充值（难度 3）
- Shiro 安全组件（难度 3）
- 移动端适配
- 密码找回
- 多语言支持

## 3. 详细需求

### 3.1 F-01/F-02 注册与登录

**注册：**
- 用户名 2-50 字符，不可重复
- 密码 bcrypt 加密，至少 6 位
- 注册成功自动登录，返回 JWT token
- 所有新注册用户 role=user

**登录：**
- 用户名+密码验证
- 成功返回 JWT token（24h 有效）
- 前端存 token 到 localStorage，后续请求自动携带

**管理员：**
- 系统启动时自动创建（账号密码由 .env 配置）
- role=admin，唯一管理员
- 普通用户无法通过注册成为管理员

### 3.2 F-03 取名表单

| 字段 | 类型 | 必填 | 约束 |
|------|------|------|------|
| 姓氏 | 文本 | 是 | 1-5 字 |
| 性别 | 单选 | 是 | 男/女 |
| 生日 | 日期 | 否 | YYYY-MM-DD |
| 时辰 | 下拉选择 | 否 | 十二时辰 |
| 风格 | 文本 | 否 | ≤50 字 |
| 期望 | 文本 | 否 | ≤200 字 |

### 3.3 F-04/F-05 AI 生成与展示

- System Prompt：取名大师角色 + JSON 格式约束
- 每个名字含：full_name / meaning / wuxing / source
- Apple 极简风格展示：毛玻璃卡片，stagger 淡入动画

### 3.4 F-06 对话微调

- 展示为对话气泡（用户蓝色 / AI 灰色）
- 携带原始需求+对话历史发送给 DeepSeek
- 结果实时更新

### 3.5 F-07 取名历史

- 登录后可见
- 按时间倒序展示
- 点击展开查看完整名字列表
- 需要登录，只能看自己的

### 3.6 F-08 管理后台

- 仅管理员可见
- 查看所有用户的认证日志（注册/登录/登出/IP/时间）
- 查看所有用户的取名记录
- 统计卡片：总用户数、取名次数

### 3.7 F-09 错误处理

| 错误场景 | 用户提示 |
|----------|----------|
| 未登录访问需登录接口 | 跳转登录页 |
| Token 过期 | "登录已过期，请重新登录" |
| 非管理员访问 /admin | 跳转主页 |
| DeepSeek 超时 | "AI 服务响应超时，请重试" |
| JSON 解析失败 | "AI 返回格式异常，请重试" |
| 用户名已存在 | "用户名已被注册" |
| 用户名或密码错误 | "用户名或密码错误" |

## 4. 非功能需求

| 属性 | 要求 |
|------|------|
| 安全性 | 密码 bcrypt 哈希，JWT 认证，角色权限控制 |
| 可用性 | Apple 极简风格，3 步取名 |
| 响应时间 | 首次生成 <35s（含 LLM） |
| 兼容性 | Chrome/Edge/Safari 最新版 |
| 数据持久化 | SQLite（开发）/ MySQL（生产） |

---

## 5. 验收标准

1. 用户可注册、登录、登出
2. 登录后可填写表单并成功生成名字
3. 名字含寓意、五行、出处
4. 可通过对话微调结果
5. 历史页面可查看过往取名记录
6. 管理员可在后台查看认证日志
7. 未登录访问需登录接口跳转登录页
8. 普通用户无法访问管理后台
9. 网络异常时有友好错误提示

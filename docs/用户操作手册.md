# 用户操作手册 — AI 智能取名系统

> 版本：2.0 | 日期：2026-07-03

---

## 1. 开始使用

### 1.1 打开系统

浏览器访问系统地址（开发环境：`http://localhost:5173`）。

### 1.2 注册账号

首次使用需注册：

1. 点击"去注册"
2. 输入用户名（2-50 字）和密码（至少 6 位）
3. 点击"注册"
4. 注册成功后自动登录，进入取名主页

### 1.3 登录

已有账号：

1. 输入用户名和密码
2. 点击"登录"

---

## 2. 取名

### 2.1 填写信息

| 字段 | 必填 | 说明 |
|------|------|------|
| 姓氏 | 是 | 孩子的姓，1-5 字 |
| 性别 | 是 | 男或女 |
| 生日 | 否 | 格式 YYYY-MM-DD |
| 时辰 | 否 | 十二时辰下拉选择 |
| 风格 | 否 | 如"文雅书香"、"大气磅礴" |
| 期望 | 否 | 自由描述，200 字以内 |

### 2.2 生成名字

点击「生成名字」按钮，等待 5-30 秒。加载时显示骨架屏动画。

### 2.3 查看结果

每张卡片展示：
- **名字**：28px 加粗
- **寓意**：约 50 字解释
- **五行**：五行属性
- **出处**：诗文典故来源

---

## 3. 微调名字

不满意结果时，在底部输入框提意见：

```
"太文艺了，想要大气一点的"
"希望名字更阳刚"
"不要出现涵这个字"
"能结合诗经取名吗"
```

按 Enter 或点发送。AI 根据你的反馈重新生成名字。对话记录以气泡形式展示。

---

## 4. 取名历史

右上角点击「历史记录」，查看过往所有取名结果：

- 按时间倒序排列
- 点击某条记录展开查看完整名字列表
- 支持重新查看每个名字的寓意、五行、出处

---

## 5. 管理后台（管理员）

管理员登录后右上角出现「管理后台」入口：

- **认证日志**：查看所有用户的注册、登录、登出记录（含时间和 IP）
- **取名记录**：查看所有用户的取名历史
- **统计卡片**：总用户数、取名次数、今日活跃

---

## 6. 退出登录

右上角「退出登录」按钮。退出后跳转登录页，需重新登录才能使用。

---

## 7. 常见问题

**Q：忘记密码？**

A：当前版本不支持密码找回。请联系管理员重置。

**Q：可以成为管理员吗？**

A：管理员由系统预设。普通用户无法通过注册成为管理员。

**Q：历史记录能保存多久？**

A：永久保存，除非数据库被清空。

**Q：支持手机访问吗？**

A：当前版本主要适配桌面端。

**Q：生成的名字是唯一的吗？**

A：每次生成可能不同，AI 保持一定的创意性。
